一、无线电基础原理

1.1 什么是无线电波

无线电波是一种电磁波，由交变电场和磁场组成，在空间中传播时不需要介质。它的传播速度等于光速，约为 3×10⁸ 米/秒。无线电波的频率范围通常从3kHz到300GHz，涵盖了从极低频到毫米波的广阔频谱。

核心公式

无线电波的三要素关系：

c = λ × f

其中：
c = 光速 ≈ 3×10⁸ m/s
λ = 波长（米）
f = 频率（赫兹）

这个公式告诉我们：频率越高，波长越短。例如，FM广播频率100MHz对应的波长约为3米，而WiFi 2.4GHz的波长仅约12.5厘米。

1.2 无线电频谱划分

国际电信联盟（ITU）将无线电频谱划分为多个波段，每个波段有不同的特性和用途：

二、无线电系统组成

2.1 发射机原理

无线电发射机的核心任务是将信息（声音、数据、图像）加载到高频载波上，并通过天线辐射出去。这个过程称为调制。

发射机基本组成

  1. 振荡器：产生稳定的高频载波信号
  2. 调制器：将基带信号加载到载波上
  3. 功率放大器：提升信号功率至发射所需电平
  4. 天线：将电信号转换为电磁波辐射
  5. 滤波器：抑制谐波和杂散发射

2.2 调制技术详解

幅度调制（AM）

通过改变载波的振幅来传递信息。AM广播（530-1700kHz）和航空通信（118-136MHz）使用这种技术。优点是电路简单，缺点是抗干扰能力差，功率效率低。

s(t) = Ac[1 + m·x(t)]·cos(2πfct)

m = 调制指数（0 < m ≤ 1）
x(t) = 基带信号（|x(t)| ≤ 1）

频率调制（FM）

通过改变载波的频率来传递信息。FM广播（88-108MHz）和业余无线电对讲机常用。抗干扰能力强，音质好，但占用带宽较大。

s(t) = Ac·cos[2πfct + 2πkf∫x(τ)dτ]

kf = 频偏常数（Hz/V）
最大频偏 Δf = kf·|x(t)|max

数字调制

现代通信系统普遍采用数字调制，将数据转换为离散的信号状态：

调制方式调制参数状态数频谱效率应用场景

FSK频率2中蓝牙、Zigbee
PSK相位2/4/8中GPS、WiFi
QAM幅度+相位16/64/256高4G/5G、WiFi6
OFDM多载波多子载波很高5G、WiFi6、DVB

三、天线技术

3.1 天线基本原理

天线是无线电系统的"嘴巴"和"耳朵"，负责电磁波与电信号的相互转换。天线的性能由多个参数决定：

天线关键参数

  • 增益（Gain）：天线在特定方向的辐射强度与理想点源天线的比值，单位dBi
  • 方向性：天线辐射能量的空间分布特性
  • 阻抗（Z）：通常为50Ω或75Ω，需与馈线匹配
  • 驻波比（VSWR）：反映阻抗匹配程度，理想值为1:1，通常要求<1.5:1
  • 带宽：天线能有效工作的频率范围

3.3 DIY八木天线制作

八木天线是业余无线电爱好者最常自制的定向天线。以下是一个针对430MHz（70cm业余频段）的设计：

# 八木天线设计参数（430MHz） # 使用铝管或铜线制作 设计频率: f = 430 MHz 波长: λ = c/f = 300/430 ≈ 0.697 m = 697 mm 单元长度计算: 反射器: Lr = 0.495 × λ = 345 mm 有源振子: Ld = 0.473 × λ = 330 mm # 折合振子 引向器1: L1 = 0.440 × λ = 307 mm 引向器2: L2 = 0.435 × λ = 303 mm 引向器3: L3 = 0.430 × λ = 300 mm 单元间距: 反射器-振子: S1 = 0.125 × λ = 87 mm 振子-引向器1: S2 = 0.125 × λ = 87 mm 引向器间距: S3 = 0.150 × λ = 105 mm 支撑: 20mm PVC管或木条 振子: 6-10mm 铝管或粗铜线

制作要点

  1. 所有单元必须与支撑杆绝缘（使用塑料卡扣）
  2. 有源振子中心需要断开，连接馈线
  3. 使用50Ω同轴电缆（如RG-58或更好）
  4. 制作完成后用天线分析仪测量驻波比
  5. 如果SWR偏高，微调振子长度（每次2-3mm）

四、传播特性与电离层

4.1 传播方式分类

地波传播

沿地球表面传播，频率越低绕射能力越强。AM中波广播（MF频段）主要依靠地波覆盖本地范围，白天传播距离约100-300公里。

天波传播（电离层反射）

短波（HF，3-30MHz）可被电离层反射回地面，实现数百至数千公里的远距离通信。这是业余无线电最迷人的传播方式。

电离层分层

  • D层（60-90km）：白天存在，吸收低频信号，夜间消失
  • E层（90-150km）：可反射VHF信号，sporadic E突发时可通联数千公里
  • F1层（150-250km）：夏季白天存在
  • F2层（250-400km）：最高最重要层，夜间合并为F层

4.2 太阳活动与传播

电离层的电离程度受太阳辐射影响，呈现11年周期性变化：

太阳活动期特点对HF传播影响最佳频段
太阳活动高年黑子数多，耀斑频繁F2层电离强，高频段开通14-28 MHz
太阳活动低年黑子数少，电离弱低频段更稳定，高频段关闭3.5-7 MHz

传播预测工具

推荐以下网站实时查看电离层状态和传播预测：

  • DXWatch.com - 实时DX传播报告

五、业余无线电入门

5.1 什么是业余无线电

业余无线电（Amateur Radio）是一项由国家无线电管理机构批准的、供业余无线电爱好者进行自我训练、相互通信和技术研究的无线电通信业务。全球约有300万持证的业余无线电操作员。

5.2 中国业余无线电执照

在中国，操作业余无线电台需要取得业余无线电台执照和操作证书：

操作类别允许频率最大功率考试难度
A类30MHz以下 + 50-54MHz + 144-148MHz + 430-440MHz25W基础（法律法规+基础理论）
B类A类 + 所有HF频段100W中等（增加电路、传播知识）
C类所有业余频段1000W高级（全面技术知识）

考证流程

  1. 在当地无线电管理机构网站报名（通常每年2-4次考试）
  2. 自学或参加培训班（推荐教材：《业余无线电通信》）
  3. 参加理论考试（A类30题答对25题及格）
  4. 考试合格后申请呼号（如 BG1AAA、BH4XXX）
  5. 购买设备并办理电台执照

六、实践应用

6.1 VHF/UHF FM中继通联

中继台（Repeater）是架设在高山或高楼的自动转发设备，可以大大扩展手持对讲机的通信范围。

# 典型中继台参数设置 接收频率: 439.460 MHz # 电台发射，中继接收 发射频率: 434.460 MHz # 电台接收，中继发射 频差: -5 MHz (下差) 亚音: CTCSS 88.5 Hz # 防止误触发 操作步骤: 1. 将对讲机设为中继模式 2. 输入接收频率 439.460 MHz 3. 设置频差方向为 "-"，差值 5 MHz 4. 设置亚音编码 88.5 Hz 5. 按住PTT讲话，松开收听

6.2 HF短波DX通联

DX（远距离）通联是业余无线电的终极乐趣。以下是首次HF通联的完整流程：

首次HF通联 checklist

  1. 查看传播预测，选择当前开通的频段（如14MHz白天，7MHz傍晚）
  2. 架设天线（水平偶极子或端馈半波天线）
  3. 用天线分析仪调谐，确保SWR < 1.5:1
  4. 打开电台，调节音量，选择USB或LSB模式
  5. 在频段内寻找信号（CQ呼叫或回答他人的CQ）
  6. 标准呼叫格式：CQ CQ CQ, this is Bravo Hotel 4 X-ray Yankee Zulu, BH4XYZ calling CQ and standing by
  7. 对方回答后交换信号报告（RS/RST）和QTH（位置）
  8. 记录通联日志（时间、频率、呼号、信号报告、内容摘要）

七、法规与安全

7.1 中国无线电管理条例要点

重要法规

  • 无证发射违法：擅自设置、使用无线电台（站）的，可处最高50万元罚款
  • 频率许可：业余频段内操作也需持有操作证书和电台执照
  • 功率限制：严格按执照等级使用最大功率，不得擅自加大功率
  • 禁止内容：不得发送与业余业务无关的信息（如商业广告、加密通信）
  • 应急通信：发生重大自然灾害时，业余电台可参与应急通信，事后需报告

7.2 电磁安全

射频辐射超过一定功率密度会对人体产生热效应。安全操作建议：

安全距离参考

  • VHF/UHF（<50W）：天线2米内避免长时间停留
  • HF（<100W）：天线3米内避免长时间停留
  • HF（>500W）：需要专业电磁环境评估
  • 发射时不要触摸天线，可能产生射频灼伤
  • 使用定向天线时，注意主瓣方向不要对准人员聚集区

八、进阶探索方向

掌握基础后，你可以向以下方向深入：

8.1 软件定义无线电（SDR）

使用软件处理射频信号，硬件仅需简单的射频前端。推荐入门设备：RTL-SDR（¥80-150），可接收100kHz-1.75GHz信号，配合SDR#或GQRX软件即可探索频谱。

8.2 卫星通信

业余卫星（OSCAR）和ISS国际空间站都有业余无线电载荷。使用八木天线和双段电台即可与卫星通联，甚至通过ISS的数字中继发送APRS位置信息。

8.3 自制设备（QRP）

QRP指低功率（通常<5W）通信。DIY一个简单的晶体管发射机或Arduino控制的CW收发信机，体验从零开始创造通信设备的乐趣。

8.4 应急通信（EMCOMM）

加入当地业余无线电应急通信组织（如ARES、RACES），在灾害导致常规通信中断时提供志愿服务。

推荐学习资源
